# Frontend (Web)

## Core Functionalities

### Wallet Connect
- Integration with Solana wallets to enable users to connect their wallets securely.

### User Settings
- Allow users to configure:
  - **Trade Size**: Set the desired trade size for transactions.
  - **Fee Pool Settings**: Define minimum and target balances for the fee pool.
  - **Enable/Disable**: Toggle the profile's active status.

### Deposit/Withdraw UI
- Provide an intuitive interface for users to:
  - Deposit funds into their vaults.
  - Withdraw funds from their vaults.

### Status Displays
- Show real-time information about:
  - **Vault Balances**: Display the current balances in the SOL and USDC vaults.
  - **Fee Pool**: Show the current balance of the fee pool.
  - **Last Execution**: Indicate the timestamp of the last executed transaction.
  - **PnL (Profit and Loss)**: Fetch and display profit and loss data from the indexer.
